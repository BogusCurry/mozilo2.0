Da die korrekte Klammerung essentiell / obligatorisch ist, kannst Du sie prüfen indem Du den Cursor auf eine einleitende oder abschliessende Klammer setzt - Das entsprechende Gegenstück wird dann automatisch farblich hervorgehoben z.B. [ELEMENT| ]   [ELEMENT| ]

Umschliessende Elemente

Das bedeutet das selektierter / ausgewählter Text bei Mausklick auf das Element von diesem umschlossen / umklammert wird.
Ist kein Text selektiert wird es einfach an der aktuellen Cursorposition eingesetzt und kann so ggf. mit Inhalt ergänzt werden.


Ersetzende Elemente

Das bedeutet das selektierter / ausgewählter Text bei Mausklick auf das Element von diesem ersetzt wird.
Ist kein Text selektiert wird es einfach an der aktuellen Cursorposition eingesetzt.


Wenn kein Text selektiert wurde und ein umschliessendes Element wird gewählt, kann entweder sofort weiterer Inhalt eingegeben werden oder auch ein zusätzliches umschliessendes Element gewählt werden, da sich der Cursor bereits an der richtigen Position befindet ("Verschachtelung").
