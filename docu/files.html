Galerien, also Bilderserien/-sammlungen zu einem Thema werden separat im adminTab "Galerien" verwaltet.

Mit Erstellung einer neuen Kategorie im admin Tab "Kategorien/Inhaltsseiten" wird hier automatisch ein Ordner für die zugehörigen Dateien angelegt.
Icon Legende
{ICON_edit} Öffnet und schliesst den Inhalt mit der Möglichkeit Dateien hochzuladen, zu löschen, umzubenennen oder zur Vorschau aufzurufen bzw. zu öffnen.

Hier werden Dateien verwaltet, welche zum Download angeboten oder allgemein innerhalb der Website verwendet werden können (z.B. Einzelbilder, PDF-Dokumente, MP3-/MP4-Dateien, etc.).
Jedes Feld steht für eine Kategorie. Reine Link-Kategorien (ohne Inhaltsseiten) werden hier nicht angezeigt.

Dateien können auch (mehrfach) in anderen Inhaltsseiten anderer Kategorien verwendet werden.

Mit einem Mausklick auf das jeweilige Dateityp Icon öffnet browserabhängig entweder ein Fenster, in dem Dir der Dateiinhalt angezeigt oder die Datei zum Dowload angeboten wird.
