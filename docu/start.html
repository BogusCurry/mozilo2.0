Weil Dein CMS auf dem Webserver installiert ist, und im Wesentlichen / hauptsächlich online mit dem Browser gearbeitet wird, ist es nötig die geänderten Daten an den Webserver zu übermitteln.
Dies geschieht automatisch durch Mausklick auf einen Button oder eine (Checkbox) ausser bei (Textfeldern) - hier muss die Texteingabe mit der Eingabetaste / Enter bestätigt werden.
Während der Datenübertragung an den Webserver wird ein Dialogfenster geöffnet, dass nach Ende der Übertragung automatisch wieder geschlossen wird. Dieser Vorgang kann so schnell ablaufen, dass er unbemerkt somit unsichtbar abläuft, kann aber bei (langer) Anzeige als Indikator für langsame Verbindungen oder Verbindungsprobleme dienen.

Dein CMS arbeitet dateibasiert, d.h. bedeutet jede Kategorie obliegt / entspricht einem Ordner auf dem Webserver, jede Inhaltseite einer Textdatei in selbigem. Hochgeladene Dateien werden im jeweiligen Kategorieordner in einem Unterordner namens "dateien" auf dem Server gespeichert / abgelegt. Auch Galerien sind Ordner auf dem Webserver in dem die hochgeladenen Bilder abgelegt sind (vgl. Kategorie / Inhaltsseiten). Vorschaubilder werden im jeweiligen Galerieordner in einem Unterordner namens "vorschau" auf dem Server erzeugt.

Verzeichnisstruktur / -baum des CMS 
admin/                  Enthält alle Funktionen die zur Bearbeitung / zum Ändern der Website benötigt werden.
      conf/                 Enthält die zum admin-Verzeichnis gehörigen Konfigurationsdateien / Einstellungen.
cms/                    Enthält alle Funktionen die zur Ausgabe / zum Ändern der Website benötigt werden.
    conf/                   Enthält die zum cms-Verzeichnis gehörigen Konfigurationsdateien / Einstellungen.
kategorien/             Hier befinden sich die Kategorien, Inhaltseiten und Dateien.
         Eine_Kategorie/
                        dateien/
                                Eine_Datei
                        Eine_Inhaltsseite
galerien/               Hier befinden sich die Galerien und Bilder.
         Eine_Galerie/
                       vorschau/
                                Die_Vorschaubilder (Werden automatisch von CMS-generiert - Kein Upload)
                       Die_Bilder
plugins/                Hier sind die installierten Plugins abgelegt.
        Die_Plugins
layouts/                Hier sind die installierten Layouts enthalten.
        Die_Layouts
docu/                   Hier ist die komplette Online-Dokumentation / Hilfe abeglegt (u.a. diese Seite).
jquery/                 Die jQuery und jQueryUI Scripte enthalten Codebibliotheken die u.a. im admin-Bereich
			und von einigen Plugins genutzt werden.
install/                Hier befindet sich der Installer, also das Setup-Programm Deines CMS.
			Dieser Ordner sollte nach erfolgter Installation gelöscht werden.

Sofern Du die Strukturrichtlinie eingehalten hast, sind für ein wiederherstellungsfähiges Backup / eine Datensicherung
einfach folgende Ordner zu sichern:

admin/conf
cms/conf
kategorien
galerien
plugins

Mit dem Download der letzten Version des CMS, anschliessendem FTP-Upload und erneutem FTP-Upload dieser Datensicherung kann Deine Website dann innerhalb kurzer Zeit wieder funktional online sein.
( Modifikationen,Hacks,db-Basierende Dienste -ausdrücklich- ausgeschlossen. ). Dies gilt inbesondere für Support / Hilfe.
