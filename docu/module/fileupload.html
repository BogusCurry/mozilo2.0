Der Dateien Manager
{IMG_file_manager}

Achtung! Alle Aktionen werden nicht nachgefragt! So bedeutet z.B. Löschen es wird sofort und ohne Nachfrage gelöscht.

Hier kannst Du Deine Dateien hochladen, verwalten / sortieren, umbennen oder löschen.

Icon Legende
{IMG_files_add} Dateien Auswahl Dateien für das Hochladen auswählen (Mehrfachauswahl browserabhängig möglich!).
{IMG_files_start} Alle Hochladen Alle ausgewählten Dateien hochladen.
{IMG_files_cancel} Alle Abbrechen Der aktuelle Hochladevorgang wird für allen laufenden Dateien abgebrochen.
{IMG_files_del} Alle Löschen Alle Dateien löschen.
{IMG_file_start} Hochladen Diese ausgewählte Datei jetzt hochladen.
{IMG_file_cancel} Abbrechen Hochladevorgang für diese ausgewählte Datei abbrechen bzw. verwerfen.
{IMG_file_del} Löschen Diese Datei löschen.

Zum Hochladen einfach auf den Dateien Auswahl Button klicken, dieser öffnet ein Dialogfenster, in dem Du hochzuladende Dateien auswählen kannst.

Browserabhängig kannst Du hier mehrere Dateien ausgewählen / markieren ("taggen", +StrG, +Shift) werden.

Die ausgewählten Dateien erscheinen dann im Dateien Manager.
Hier verfügt jede aufgeführte Datei jeweils über eigene Buttons zum Hochladen oder Abbrechen welche nur für den Upload dieser Datei gelten.

Dateien können erst umbenannt werden, wenn sie bereits hochgeladen wurden.

Mit einem Doppelklick auf den Dateinamen kannst Du diesen ändern. Mit der Eingabetaste / Enter bestätigst Du die Änderung.

Die Endung ("Extension" z.B. .jpg) einer Datei soltest Du keinenfalls ändern.
