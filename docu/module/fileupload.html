Dateitypen

Der Dateienmanager erkennt die meisten Dateitypen (z.B. Bild- oder Textdatei) anhand ihrer Dateiendung (z.B. ".jpg" für eine JPG-Bilddatei).
So sorgt der Dateienmanager aufgrund der Symboldarstellung (siehe folgende Tabelle) für einen besseren Überblick über die vorhandenen Dateien.

Dateivorschau bei Bild- und Textdateien

Der Dateienmanager kann Bild- (.jpg, .gif, .png, .bmp) und Textdateien (.txt) in einer eigenen Vorschau darstellen. Hierfür klickst Du auf das Symbol links neben dem Dateinamen (siehe Screenshot (2) weiter oben).
Um die dann angezeigte Vorschau wieder zu schließen, klickst Du rechts oben auf das "Schließen" Symbol (X) der Vorschau.

Voraussetzung für die Vorschau: Die Dateien müssen bereits hochgeladen sein.


Neue Datei(en) hochladen (Upload oder Überschreiben)

Um eine neue Datei hochzuladen, klickst Du zunächst auf das "Datei Auswahl" Symbol {IMG_files_add} in der "Datei Menüleiste", welches ein Dialogfenster öffnet, in dem Du die hochzuladende Datei(en) auswählen kannst.

Browserabhängig können hier mehrere Dateien gleichzeitig ausgewählt / markiert ("taggen", +StrG, +Shift) werden.

Die ausgewählten Dateien erscheinen dann anschließend im Dateien Manager, inkl. der Angabe der Größe der Datei (z.B. "42,30 KB").

  • Dateien einzeln hochladen:
    Hier verfügt jede aufgeführte Datei im rechten Bereich jeweils über eigene Buttons zum Hochladen {IMG_files_start} oder Abbrechen {IMG_files_cancel} (nicht hochladen).

  • Mehrere oder alle Dateien gleichzeitig hochladen:
    Nach der Auswahl von mehreren Dateien ist die Checkbox {IMG_files_checkbox} bei allen Dateien bereits mit einem Häkchen versehen. Bei Bedarf kannst Du das Häkchen bei Dateien entfernen, die nicht hochgeladen werden sollen.
    Um alle ausgewählten Dateien hochzuladen, klickst Du nun auf das Icon "Hochladen" {IMG_files_start} in der "Datei Menüleiste".
    Je nach Gesamtgröße der Datei(en) und der Internetanbindung kann der Upload einige Zeit in Anspruch nehmen.
    Um den Uploadvorgang jedoch abzubrechen, klickst Du einfach auf das Icon "Abbrechen" {IMG_files_cancel} in der "Datei Menüleiste".

  • Dateien überschreiben:
    Wenn die Datei, die hochgeladen werden soll, bereits mit exakt demselben Namen gleichen Dateityps existiert, erscheint die Meldung "Überschreibe Datei" im Dateienmanager (siehe folgenden Screenshot).
    Hierbei entscheidest Du nun, ob die bestehende Datei überschrieben werden soll (Icon "Hochladen") oder nicht (Icon "Abbrechen").

    Achtung: Das Überschreiben einer Datei kann nicht rückgängig gemacht werden!

    Screenshot (3): Warnhinweis vor dem Überschreiben einer bereits existierenden Datei
    {IMG_files_overwrite}

Tipp: Gib den Dateien bereits vor dem Hochladen eindeutige, nur einmal vorkommende Namen, um evtl. Verwechselungen zu vermeiden!

Nach erfolgreichem Upload erscheinen die Dateien im Dateienmanager (siehe Screenshot (2) weiter oben) und können nun verwendet werden.


Hochgeladene Dateien umbenennen

Der Dateienmanager erlaubt es, die Namen der Dateien ohne gesonderten FTP-Zugriff zu ändern.
Voraussetzung hierfür ist, dass die Datei bereits hochgeladen ist.

Um den Namen einer Datei zu ändern, klickst Du einfach doppelt auf den Namen der Datei (z.B. "text.txt", siehe Screenshot (2) weiter oben), sodass sich das Textfeld öffnet. Nun kannst Du den Dateinamen ändern.

Achtung: Die Extension / Dateiendung (z.B. ".jpg" oder ".txt") solltest Du auf keinen Fall ändern, da der Dateityp sonst nicht erkannt werden kann!

Zum Speichern des neuen Namens betätigst Du die Eingabetaste / Enter. Um die Änderung abzubrechen, drückst Du hingegen die ESC Taste.
Wenn der gewünschte neue Name bereits für eine andere Datei gleichen Typs existiert, erscheint eine Fehlermeldung.


Dateien löschen

Mit dem Dateienmanager können einzelne, mehrere oder alle Dateien einer Kategorie ohne gesonderten FTP-Zugang gelöscht werden.
  • Datei einzeln löschen:
    Um eine einzelne Datei zu löschen, klickst Du im rechten Bereich der betreffenden Datei auf das Löschen Symbol {IMG_files_del} (siehe Screenshot (4) "Einzelne Datei löschen" weiter unten).
    Es folgt eine Sicherheitsabfrage, bei der Du das Löschen bestätigst ("Ja") oder abbrichst ("Nein").

  • Mehrere Dateien löschen:
    Um mehrere Dateien zu löschen, klickst Du zuerst im rechten Bereich der betreffenden Dateien in die Checkbox {IMG_files_checkbox} der zu löschenden Dateien.
    Anschließend klickst Du auf das Löschen Symbol {IMG_files_del} in der Datei Menüleiste (siehe Screenshot (4) "Anwenden auf alle ausgewählten Dateien (Löschen)" weiter unten).
    Es folgt eine Sicherheitsabfrage, bei der Du das Löschen bestätigst ("Ja") oder abbrichst ("Nein").

  • Alle Dateien einer Kategorie löschen:
    Um alle Dateien einer Kategorie zu löschen, markierst Du alle Dateien per Klick auf die Checkbox {IMG_files_checkbox} in der Datei Menüleiste (siehe Screenshot (4) "Checkbox, um alle Dateien auszuwählen" weiter unten).
    Anschließend klickst Du auf das Löschen Symbol {IMG_files_del} in der Datei Menüleiste (siehe Screenshot (4) "Anwenden auf alle ausgewählten Dateien (Löschen)" weiter unten).
    Es folgt eine Sicherheitsabfrage, bei der Du das Löschen bestätigst ("Ja") oder abbrichst ("Nein").
