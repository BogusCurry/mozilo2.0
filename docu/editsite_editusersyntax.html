Achtung diese Beschreibung gilt nur für Eigene Syntaxelemente erstellen in dem admin Tab Einstelungen.

Ein Egenes Syntaxelement setzt sich aus dem Namen des Elements und dem eigenen Code zusamen.

der Name darf nur aus den Buchstaben (a-z oder A-Z), Zahlen und als einziges Sonderzeichen dem Unterstrich "_" bestehen.
Als Trennug wird " = " benutzt. Achtung die Lehrzeichen um das "=" sind Pflicht.
Der eigenen Code darf alles enthalten, auch Zeilenumbrüche sind erlaubt.
Es gibt diese speziellen Platzhalter {DESCRIPTION} und {VALUE} die nur in Eigene Syntaxelemente erstellen benutzbar sind.
{DESCRIPTION} steht für den Inhalt zwischen dem = und dem | Zeichen
{VALUE} steht für den Inhalt zwischen dem | und dem ] Zeichen

Achtung wenn eins dieser {, [, |, ], } Zeichen als Zeichen im ersetzenden Teil benutzt werden soll, muss es mit einem Hochdach "^" geschützt werden.
Es kann auch vorkommen das das "=" geschützt werden muss, oder auch andere Sonderzeichen

Beispiel:
Eigenes Syntaxelement

wikipedia = [link={DESCRIPTION}|http://de.wikipedia.org/wiki/{VALUE}]

Im Editior sieht das dann so aus

[wikipedia= | ]

wir setzen mal was ein

[wikipedia= was ist HTML | Html ]

als ergebnis erhalten wir

was ist HTML
